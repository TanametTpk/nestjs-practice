## Description

[Nest](https://github.com/nestjs/nest) is backend framework for js/ts, just like Spring boot, or other word it's React for Backend.

## Installation

```bash
$ npm i -g @nestjs/cli
$ nest new project-name
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
